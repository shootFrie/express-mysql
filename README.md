# element-demo

## 简介
使用Element-ui框架搭建博客后台，一共三个页面，创建页面，列表页面，编辑页面，实现了对数据库的增删改查功能。数据库使用mysql，前端用axios和后台数据进行交互，后台使用express

```
npm install
```
